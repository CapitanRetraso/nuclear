# Nuclear
reactor.par .bin files to JSON exporter and importer.

Games by RGG Studio that use these files: Yakuza 0, Yakuza Kiwami and Ryu ga Gotoku Ishin.


  
### Supported headers

* colb (collision.bin)
* motb (motion.bin)


  
### How to use

```"nuclear" <file>```

You can also drag&drop
